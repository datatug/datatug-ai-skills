# datatug queries — run

Execute a saved query, optionally with parameters.

## Command

```bash
datatug queries <subcommand> [args]
```

## When to use

- Ad-hoc troubleshooting using a parametrised query.
- Scripted reporting from CI or a cron job.

## Notes

- Exact subcommand names (e.g., `run`, `exec`) and parameter-passing syntax depend on the CLI version. Run `datatug queries --help` for the authoritative list.

# datatug projects add

Register a DataTug project on this machine so it can be referenced by ID.

## Command

```bash
datatug projects add <path-or-id>
```

## When to use

- After cloning a DataTug project repo from Git.
- After scaffolding a new project with [`init`](../../init/SKILL.md).

## Notes

- Run `datatug projects --help` for the authoritative subcommand list and flags.
